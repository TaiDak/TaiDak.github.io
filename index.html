#Taken as an example from Wilson Magic repo Tai Dak

Current Version: 0.0.1

Repository addon

Instructions for adding this repo:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://taidak.github.io/repo/ (Give it the name "TaiDak").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "Wilson Magic", and install repository.wilson-magic-0.0.1.zip
  • Go back to Addons install, but this time, select "Install from repository"
  • Select "TaiDak's Kodi Addon Repo"
  • Go into the "Video add-ons" section in the repo, and you'll find Hummingbird

Enjoy!
